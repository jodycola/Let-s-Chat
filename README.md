## Stack

Let's chat is a fullstack application that makes use of the following technologies  
- Ruby on Rails
  - Action Cable
  - JWT Authentication
  - Redis 
  - Websockets

- JavaScript
  - React
  - Styled-Components
  - Action Cable for React
